# Политика конфиденциальности RoboShare

Этот репозиторий содержит Политику конфиденциальности для мобильного приложения RoboShare.

## Просмотр политики

Политика конфиденциальности доступна по адресу:
- [Политика конфиденциальности (index.md)](index.md)

## Для разработчиков

После внесения изменений в политику конфиденциальности:

1. Обновите дату "Последнее обновление" в начале документа
2. Зафиксируйте изменения в репозитории
3. Убедитесь, что ссылка в Google Play Console актуальна

## GitHub Pages

Эта политика может быть опубликована через GitHub Pages:
1. Перейдите в Settings → Pages
2. Выберите Source: Deploy from a branch
3. Выберите ветку: main
4. Сохраните

После активации, политика будет доступна по адресу:
https://ateipov1-bot.github.io/privacy_policy/

## Контакты

Для вопросов о политике конфиденциальности свяжитесь с RoboSpark.
